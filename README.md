# üìÖ Dynamic JavaScript Calendar Application

This project implements a user-friendly, Google Calendar-like interface using only **HTML, CSS, and vanilla JavaScript**. It features dynamic month navigation, the ability to add new events, and sophisticated conflict detection based on event scheduling times.

---

## ‚ú® Features

* **Month Navigation:** Easily move between previous and next months.
* **Current Day Highlight:** Visually highlights the current date for context.
* **Dynamic Event Loading:** Events are loaded from a static data source (JS array).
* **Event Creation:** Users can click on any date cell to add new events via a modal form.
* **Time-Based Conflict Detection:** The calendar checks for actual **time overlap** between scheduled events on the same day.
    * **Visual Warning:** Overlapping events are color-coded (Orange border) for immediate user feedback.
